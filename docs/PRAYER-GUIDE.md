# 성무일도 기도 순서 가이드 — Залбирлын дараалал

> 이 문서는 몽골어 성무일도(시간전례) 앱에서 **아침기도와 저녁기도를 올바른 순서로 바치기 위한 안내서**입니다.
> Энэ баримт бичиг нь Монгол хэл дээрх Залбиралт цагийн ёслол аппликэйшн дахь **өглөөний болон оройн даатгал залбирлыг зөв дарааллаар уншихад** зориулагдсан юм.

---

## 1. 앱 구조 개요

앱의 콘텐츠는 3개 그룹으로 나뉩니다:

| 그룹 | 몽골어명 | 내용 | 용도 |
|------|---------|------|------|
| **시편집** (4주 시편집) | Дуулал номын дөрвөн долоо хоног | 4주 주기 시편/찬가 | 매일 기도의 **시편 부분** (고정 주기) |
| **고유문** (시기별 고유문) | Цаг улирлын онцлог шинж | 6개 전례 시기별 텍스트 | 시기에 따라 바뀌는 **독서, 응답, 마침기도** |
| **찬미가** | Магтуу | 109곡 개별 찬미가 | 기도 시작부의 **찬미가** |

**핵심 원리**: 한 번의 기도(예: 화요일 아침기도)를 바치려면 이 3개 그룹을 **교차하며** 읽어야 합니다.

---

## 2. 아침기도 (Lauds) 표준 순서 — Өглөөний даатгал залбирал

| 순서 | 전례 요소 | 몽골어 | 앱에서 찾는 방법 |
|------|----------|--------|----------------|
| **1** | 시작 기원 | Эзэн, уруулыг минь нээгээч / Тэнгэрбурхан минь, надад туслахаар соёрхоно уу | *본문 내 포함* (별도 섹션 없음, х. 44 참조) |
| **2** | 초대찬미 (시편 95) | Урих дуудлага / Дуулал 95 | 시편집에서 "Урих дуудлага" 제목 섹션 (하루 첫 기도에서만, х. 28 참조) |
| **3** | 찬미가 | Магтуу | → **찬미가 그룹**으로 이동하여 해당 곡 선택 (х. 874-879 참조) |
| **4** | 시편 1 | Дуулал + 번호 | → **시편집** 해당 주차/요일/아침기도의 첫 번째 시편 섹션 |
| **5** | 구약 찬가 | Магтаал / Шад магтаал | → **시편집** 해당 요일의 "Магтаал" 또는 "Шад магтаал" 섹션 |
| **6** | 시편 2 (찬양시편) | Дуулал + 번호 | → **시편집** 해당 요일의 두 번째 시편 섹션 (있는 경우) |
| **7** | 짧은 독서 | Уншлага | → **고유문** 현재 전례 시기 → 해당 요일의 "Өглөөний даатгал залбирал" 아래 "Уншлага" |
| **8** | 짧은 응답송 | Хариу залбирал | → **고유문** 해당 요일의 "Хариу залбирал" |
| **9** | 복음 찬가 (베네딕투스) | Захариасын магтаал | → **고유문** 해당 요일의 "Захариасын магтаал / Шад магтаал" |
| **10** | 청원 기도 | Гуйлтын залбирал | → **시편집** 또는 **고유문**에서 "Гуйлтын залбирал" 섹션 |
| **11** | 주님의 기도 | Тэнгэр дэх Эцэг минь ээ... | *암기* (본문에 포함되지 않음) |
| **12** | 마침 기도 | Төгсгөлийн даатгал залбирал | → **고유문** 해당 시기/요일의 "Төгсгөлийн даатгал залбирал" |
| **13** | 파견 | - | *본문 내 포함* (х. 44 참조) |

## 5. 페이지 참조 디코더 (х. NNN)

본문에 등장하는 `х. NNN`은 **실물 성무일도 책의 페이지 번호**를 가리킵니다. 앱에서는 다음과 같이 대응합니다:

| 페이지 참조 | 의미 | 앱 내 대응 |
|------------|------|-----------|
| **х. 28** | 초대 시편 (Урих дуудлагын дуулал, 시편 95) | 시편집에서 "Урих дуудлага" 제목 섹션 참조. 일부 요일에 본문 포함됨 |
| **х. 38** | 저녁기도 마침 순서 (Төгсгөлийг дэг жаягийн дагуу дуусгана) | 앱에 별도 텍스트 없음. 마침기도는 고유문에서 찾을 것 |
| **х. 44** | 아침기도 마침 순서 | 앱에 별도 텍스트 없음. 마침기도는 고유문에서 찾을 것 |
| **х. 874-875** | 찬미가 섹션 시작 (Магтуу) | → **찬미가 그룹** 전체 (109곡) |
| **х. 876** | 특정 찬미가 페이지 | → **찬미가 그룹**에서 제목으로 검색 |
| **х. 878-879** | 특정 찬미가 페이지 | → **찬미가 그룹**에서 제목으로 검색 |
| **х. 544-545** | 성모 마리아 대송 (Мариагийн антифон) | 앱에 별도 텍스트 없음 |

> **참고**: х. 28, х. 38, х. 44는 실물 책 앞부분의 고정 기도문으로, 현재 앱에 파싱되지 않았습니다.

## 6. 전례 시기별 고유문 사용법

고유문은 전례 시기에 따라 독서, 응답송, 마침기도 등을 제공합니다.

### 6.1 전례 시기 (Цаг улирал)

| 시기 | 몽골어 | 앱 경로 | 대략적 기간 |
|------|--------|--------|-----------|
| 대림 | Ирэлтийн цаг улирал | 고유문 → 대림 | 11월 말 ~ 12월 24일 |
| 성탄 | Эзэний мэндлэлтийн цаг улирал | 고유문 → 성탄 | 12월 25일 ~ 주님 세례 축일 |
| 사순 | Дөчин хоногийн цаг улирал | 고유문 → 사순 | 재의 수요일 ~ 부활 전야 |
| 부활 | Амилалтын цаг улирал | 고유문 → 부활 | 부활 주일 ~ 성령 강림 후 |
| 연중 | Жирийн цаг улирал | 고유문 → 연중 | 나머지 기간 |
| 성인축일 | Гэгээнтнүүдийн | 고유문 → 성인 | 특정 축일 |

### 6.2 고유문에서 찾는 법

1. **사이드바** → 시기별 고유문 (Цаг улирлын онцлог шинж)
2. 현재 전례 시기 선택 (예: 대림)
3. **해당 요일** 선택 (예: ДАВАА ГАРАГ = 월요일)
4. 본문에서 필요한 부분을 찾기:
   - `Өглөөний даатгал залбирал` = 아침기도용
   - `Оройн даатгал залбирал` = 저녁기도용 (1/2 дугаар = 제1/제2 저녁기도)
   - `Уншлага` = 짧은 독서
   - `Хариу залбирал` = 짧은 응답송
   - `Шад магтаал` = 복음 찬가 후렴
   - `Төгсгөлийн даатгал залбирал` = 마침 기도

### 6.3 연중 시기 주일 고유문

연중 시기에는 주일마다 별도 고유문이 있습니다:
- 고유문 → 연중 → "1주일" ~ "34주일" (1-34 дахь ням гараг)
- 해당 주일의 마침기도, 복음 찬가 후렴 등을 여기서 가져옵니다

## 7. 주간 시편집 구조

시편집은 4주 주기로 반복됩니다. 각 주차는 토요일 저녁기도(일요일 제1저녁기도)부터 시작합니다.

### 요일별 기도 요소 (1주차 기준)

| 요일 | 아침기도 요소 | 저녁기도 요소 |
|------|-------------|-------------|
| 토 | 시편 119 → (계속) → 독서 → 마침기도 | 시편 141 → 찬가 → 독서 → (계속) |
| 일 | 마리아찬가 → 청원기도 → 찬가 → (계속) → 시편 149 → 독서 | (계속) → 시편 110 → 시편 114 → 찬가 → 찬가 → 초대찬미 |
| 월 | 시편 5 → 찬가 → 독서 → 마침기도 | (계속) → 시편 15 → 응답 |
| 화 | 초대찬미 → 찬가 → 시편 33 → (계속) → 응답 | 시편 20 → 시편 21 → 찬가 → 응답 |
| 수 | 시편 36 → 찬가 → 시편 47 → 마침기도 | (계속) → 찬가 → 독서 → 마리아찬가 |
| 목 | 시편 57 → 찬가 → (계속) → 독서 | (계속) → 시편 32 → 찬가 → 마리아찬가 |
| 금 | 초대찬미 → 시편 51 → 찬가 → 시편 100 → 청원기도 | 시편 46 → 찬가 → 독서 → 마침기도 |

> **참고**: "(계속)"으로 표시된 부분은 앱에서 제목 없이 나타나는 섹션입니다 (이전 섹션의 연속).

## 8. 실전 팁

### 기도 시작 전 확인사항
1. **오늘이 몇 주차인지** 확인 (1~4주차 반복)
2. **현재 전례 시기** 확인 (대림/성탄/사순/부활/연중)
3. **아침기도인지 저녁기도인지** 확인

### 기도 흐름 요약 (간편 버전)

```
1. 시작 기원 (암기 또는 본문 첫 줄)
2. [첫 기도만] 초대찬미 — 시편집에서 "Урих дуудлага"
3. 찬미가 — 찬미가 그룹에서 선택
4. 시편 + 찬가 — 시편집에서 해당 주차/요일 순서대로
5. 짧은 독서 — 고유문에서 현재 시기/요일
6. 짧은 응답송 — 고유문에서 현재 시기/요일
7. 복음 찬가 — 아침: 베네딕투스, 저녁: 마니피캇
   (시편집 또는 고유문에서 후렴 확인)
8. 청원 기도 — 시편집 또는 고유문
9. 주님의 기도 (암기)
10. 마침 기도 — 고유문에서 현재 시기/요일
```

### 교차 이동이 필요한 순간

기도 중 **그룹을 전환해야 하는 시점**:

| 시점 | 이동 방향 | 이유 |
|------|----------|------|
| 3단계 (찬미가) | 시편집 → **찬미가** | 찬미가는 별도 그룹에 있음 |
| 5-6단계 (독서/응답) | 시편집 → **고유문** | 시기별 독서는 고유문에 있음 |
| 10단계 (마침기도) | 시편집 → **고유문** | 시기별 마침기도는 고유문에 있음 |

> **팁**: 기도 전에 고유문의 해당 요일 섹션을 먼저 열어두면 교차 이동이 편합니다.
