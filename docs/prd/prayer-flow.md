## 기능 목록

### F-FLOW: 기도 흐름 가이드

| 항목 | 내용 |
|------|------|
| **ID** | F-FLOW |
| **이름** | 기도 흐름 가이드 |
| **상태** | 진행중 |
| **우선순위** | P1 (중) |
| **설명** | 한 기도 세션(예: 아침기도)에서 필요한 섹션들을 순서대로 안내하는 기능. 현재 사용자가 직접 어떤 섹션을 읽어야 하는지 알아야 함. "이전 위치로 돌아가기" 버튼, 이동 이력 스택 등 포함. |
| **관련 시나리오** | S1 (아침기도 — 교차 그룹 이동 후 복귀 불가), S9 (저녁기도 — 4회 그룹 간 이동) |
| **관련 문서** | [`docs/PRAYER-GUIDE.md`](../PRAYER-GUIDE.md) — 아침/저녁기도 표준 순서 |

**핵심 문제:**
- 1회 기도에 **그룹 간 이동 3~4회** (시편집→찬미가→시편집→고유문→시편집)
- 매번 **사이드바 수동 탐색** 필요 (3~5번 클릭)
- **복귀 시 이전 위치 유실** — 어디까지 읽었는지 기억해야 함

## 기도 흐름 아키텍처 (설계 방향)

### 아침기도 (Lauds) 표준 순서

```
1. 시작 기원 (암기/본문)
2. [첫 기도만] 초대찬미 — 시편집
3. 찬미가 — 찬미가 그룹 ← 교차 이동
4. 시편 + 찬가 — 시편집
5. 짧은 독서 — 고유문 ← 교차 이동
6. 짧은 응답송 — 고유문
7. 복음 찬가 (베네딕투스) — 고유문
8. 청원 기도 — 시편집/고유문
9. 주님의 기도 (암기)
10. 마침 기도 — 고유문 ← 교차 이동
```

### 교차 이동 시점

| 시점 | 이동 방향 | 이유 |
|------|----------|------|
| 3단계 | 시편집 → **찬미가** | 찬미가는 별도 그룹 |
| 5-6단계 | 시편집 → **고유문** | 시기별 독서는 고유문 |
| 10단계 | 시편집 → **고유문** | 시기별 마침기도는 고유문 |
