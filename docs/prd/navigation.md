## 기능 목록

### F-NAV: 계층 목차 네비게이션

| 항목 | 내용 |
|------|------|
| **ID** | F-NAV |
| **이름** | 계층 목차 네비게이션 |
| **상태** | ✅ 구현 완료 |
| **설명** | 478개 노드(최대 depth 5)의 트리뷰 목차. 자동 펼치기, 활성 항목 하이라이트, 사이드바 자동 스크롤 지원. 데스크탑은 정적 사이드바, 모바일은 오버레이 방식. |
| **관련 시나리오** | S1 (아침기도 탐색), S2 (전례 준비 트리 탐색), S4 (찬미가 탐색), S5 (첫방문 트리 탐색), S8 (시기 전환 트리 탐색) |

### F-SEARCH: 텍스트 검색

| 항목 | 내용 |
|------|------|
| **ID** | F-SEARCH |
| **이름** | 텍스트 검색 |
| **상태** | ✅ 구현 완료 |
| **설명** | 제목 기반 검색 (2자 이상 입력). 검색 결과 클릭 시 해당 섹션으로 이동 + 검색 초기화. 현재 제목만 검색 가능하며 본문 검색은 F-FULLSEARCH로 별도 관리. |
| **관련 시나리오** | S4 (찬미가 번호 검색), S6 (시편 번호 검색) |

### F-BREAD: 브레드크럼

| 항목 | 내용 |
|------|------|
| **ID** | F-BREAD |
| **이름** | 브레드크럼 |
| **상태** | ✅ 구현 완료 |
| **설명** | 현재 위치를 경로로 표시 (예: "4주 시편집 / 1주차 / 토요일 / 저녁기도"). 네비게이션 후 자동 업데이트. |
| **관련 시나리오** | S2 (전례 준비 위치 확인), S6 (시편 위치 확인) |

### F-A11Y: 접근성

| 항목 | 내용 |
|------|------|
| **ID** | F-A11Y |
| **이름** | 접근성 |
| **상태** | ✅ 구현 완료 |
| **설명** | ARIA tree roles, 키보드 네비게이션, sr-only 라벨. WCAG 2.1 AA 기준 준수. 44px 최소 터치 타겟. |
| **관련 시나리오** | 전체 시나리오에 간접 영향 |

### F-XNAV: 교차 그룹 네비게이션

| 항목 | 내용 |
|------|------|
| **ID** | F-XNAV |
| **이름** | 교차 그룹 네비게이션 |
| **상태** | 예정 |
| **우선순위** | P0 (높음) |
| **설명** | 본문 내 페이지 참조(`х. NNN`)가 현재 plain text로 클릭 불가. 이를 클릭 가능한 링크로 변환하여 시편집↔고유문↔찬미가 간 원클릭 이동 지원. 기도 중 교차 그룹 이동이 수동 사이드바에 의존하는 문제 해결. |
| **관련 시나리오** | S1 (아침기도 교차 이동), S9 (저녁기도 교차 이동) |

| 항목 | 내용 |
|------|------|
| **ID** | F-KEYS |
