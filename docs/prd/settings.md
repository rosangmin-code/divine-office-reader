# 모듈 PRD: settings

> **모듈**: settings
> **담당 영역**: 폰트 크기 조절, 다크 모드, 설정 영속성, iOS viewport
> **관련 UX 시나리오**: S2, S3, S5

---

## 기능 목록

### F-FONT: 폰트 크기 조절

| 항목 | 내용 |
|------|------|
| **ID** | F-FONT |
| **이름** | 폰트 크기 조절 |
| **상태** | ✅ 구현 완료 |
| **설명** | 14-32px 범위, ±2px 단위 조절. 툴바의 A+/A- 버튼으로 조작. 설정값은 localStorage에 영속. |
| **관련 시나리오** | S2 (전례 준비 시 확대), S3 (야간 기도 큰 글씨) |

### F-DARK: 다크 모드

| 항목 | 내용 |
|------|------|
| **ID** | F-DARK |
| **이름** | 다크 모드 |
| **상태** | ✅ 구현 완료 |
| **설명** | 토글 버튼으로 다크/라이트 전환. localStorage에 영속. FOUC(Flash of Unstyled Content) 방지를 위한 인라인 스크립트 적용. `html.dark` 클래스 기반. |
| **관련 시나리오** | S3 (야간 기도 다크모드), S5 (첫방문 다크모드 토글), S9 (저녁기도 다크모드) |

### F-PERSIST: 설정 영속성

| 항목 | 내용 |
|------|------|
| **ID** | F-PERSIST |
| **이름** | 설정 영속성 |
| **상태** | ✅ 구현 완료 |
| **설명** | 마지막 읽기 위치, 폰트 크기, 다크모드 설정을 `divine-office-reader-settings` localStorage 키에 저장. 앱 재방문 시 자동 복원. |
| **관련 시나리오** | S1 (아침기도 마지막 위치 복원), S3 (야간 기도 설정 유지) |
| **단위 테스트** | `reader-store.test.ts` (5개) |

### F-DVHL: iOS viewport

| 항목 | 내용 |
|------|------|
| **ID** | F-DVHL |
| **이름** | iOS viewport (100dvh) |
| **상태** | ✅ 구현 완료 |
| **설명** | iOS Safari에서 주소창 높이를 고려한 `100dvh` viewport 단위 지원. CSS 속성으로 구현되어 E2E 직접 검증은 어려움. |
| **관련 시나리오** | S3 (태블릿 야간 기도) |
