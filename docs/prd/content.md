## 기능 목록

### F-LAZY: Lazy Loading

| 항목 | 내용 |
|------|------|
| **ID** | F-LAZY |
| **이름** | Lazy Loading |
| **상태** | ✅ 구현 완료 |
| **설명** | 6개 콘텐츠 그룹(week1~4, propers, hymns)을 분할 JSON으로 관리. 스크롤 기반 pre-fetch + Route Handler(`/api/content/[group]`) + in-flight dedup으로 필요한 그룹만 로드. |
| **관련 시나리오** | S1 (아침기도 그룹 로드), S2 (전례 준비 propers 로드), S4 (찬미가 hymns 로드), S5 (첫방문 week1 자동 로드), S8 (시기 전환 그룹 전환) |
